# NodeSeek 助手 v1.2：签到 + 评论草稿

## 功能

### 签到
- 访问 NodeSeek 时自动尝试签到
- 可选每日定时打开 NodeSeek 页面签到
- 使用页面上下文执行请求，登录态由 Edge 管理
- 遇到风险提示会自动暂停

### 评论草 稿助手
- 只在 `https://www.nodeseek.com/post-*` 帖子页面显示
- 自动识别 `postId`
- 从随机评论池里选择一句
- 可自动填入页面评论框
- 发送前必须由你点击页面浮窗里的“确认发送”
- 支持每天草稿数量上限
- 支持每天确认发送数量上限
- 支持允许发送时间段
- 同一帖子当天只处理一次
- 遇到 `high risk action`、403、429 会自动暂停

## 重要说明

这个版本不是全自动批量发评论脚本。  
它会生成和填入评论草稿，但提交评论必须由你手动确认。

## 安装方法

1. 解压 zip
2. 打开 Edge：`edge://extensions/`
3. 开启“开发人员模式”
4. 建议先删除旧版 NodeSeek 签到插件
5. 点击“加载解压缩的扩展”
6. 选择解压后的文件夹：`nodeseek-edge-helper-v1.2`
7. 在 Edge 里正常登录 NodeSeek
8. 点击插件图标配置签到和评论草稿

## 使用建议

- 不要高频测试
- 不要使用明显重复的水评论
- 建议每天确认发送 1-3 条以内
- 如果出现 high risk action，先暂停几天
